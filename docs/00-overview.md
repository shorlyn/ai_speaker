# 家庭 AI 语音终端（AI Speaker）

## 目标
构建一个可长期在家庭环境运行的语音终端：
- 设备端：旧安卓手机（Flutter + Android 原生）
- 服务端：.NET 轻量网关（编排 ASR / LLM / TTS）
- 模型层：远程（Qwen / Ollama / TTS）

## 核心能力（P1）
- 手动触发录音 → 上传 → 返回语音 → 播放
- 会话记录保存
- 可更换模型而不改 App

## 非目标（P1）
- 多设备联动
- 本地大模型
- 复杂RAG/知识库
- 声纹识别

## 成功标准
1. 一句一答稳定运行
2. 连续运行24小时不崩
3. 模型替换无需改App
